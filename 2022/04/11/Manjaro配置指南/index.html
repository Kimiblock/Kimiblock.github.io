教程|如何配置 Manjaro Gnome

发表于2022-04-11|更新于2022-05-06|教程

重要事项

请到 Manjaro论坛 查询和翻阅问题

请到 ArchWiki 查阅软件指南

更换国内源

sudo pacman-mirrors -c China
sudo pacman -Syyu

中文输入法

安装软件包

终端执行

sudo pacman -Rsc fcitx
sudo pacman -S fcitx5-im fcitx5-chinese-addons fcitx5-pinyin-zhwiki

设定默认输入法

在文件管理器中, 请先打开 显示隐藏文件

在主目录 /home/你的用户名 下, 新建文件 ~/.pam_environment

打开并输入

GTK_IM_MODULE DEFAULT=fcitx
QT_IM_MODULE  DEFAULT=fcitx
XMODIFIERS    DEFAULT=\@im=fcitx
SDL_IM_MODULE DEFAULT=fcitx

启用Wayland (非必要, 高级)

打开终端, 执行

sudo rm /etc/udev/rules.d/61-gdm.rules
gsettings set org.gnome.mutter experimental-features '["kms-modifiers"]'
sudo pacman -Syu --needed xorg-xwayland libxcb egl-wayland

打开文件管理器, 导航到 /etc,

使用管理员身份编辑 mkinitcpio.conf

在 MODULES="" 中加入 nvidia nvidia_modeset nvidia_uvm nvidia_drm amdgpu

使用管理员身份编辑 /etc/default/grub

在文件的 GRUB_CMDLINE_LINUX_DEFAULT="{其他的内容}" 中加入 nvidia-drm.modeset=1

得到 GRUB_CMDLINE_LINUX_DEFAULT="nvidia-drm.modeset=1 {其他的内容}" , 在终端执行

sudo update-grub
sudo mkinitcpio -P

触控板手势 (Wayland不需要)

sudo systemctl start touchegg
sudo systemctl enable touchegg.service

修正触控板滚动速度

==Manjaro默认的滚动速度太快,需要手工设置参数==

计算参数

量出你的触控板大小, 并将量到的数字以毫米计 x0.4

写入参数

sudo pacman -S python-pip && sudo pip install libevdev pyudev
sudo libinput measure touchpad-size WxH
<!--W为计算后的长度, H为计算后的宽度-->

输入指令后用手指划过触控板边缘一圈

完成后按Ctrl-C结束, 得到数据

将类似这样的文本

------8<--------------------------------------------------
拷贝这些

------8<---------------------------------------------------

创建 /etc/udev/hwdb.d/61-evdev-local.hwdb, 将拷贝的内容粘贴进去

执行

sudo systemd-hwdb update
sudo udevadm trigger /dev/input/event*
