浏览前必读

网络与安全

连接协议

TLS

本站支持 TLSv1.2 与 TLSv1.3 且会在几年之内切换到 TLSv1.3.

IPv4 与 IPv6

站点目前不支持 IPv6 接入, 请考虑启用 6to4 技术.

由于一些原因, 运营商不支持 IPv4 IPv6 双公网接入. 这意味着本站只能在两种协议之间选择一种提供服务. 考虑到运营商对 IPv4 支持更加完善, 5年内可能都不会切换 IPv6

安全

加密

本站点使用 Nginx Https 加密, 能够最大限度保证您的浏览安全.

隐私

本网站使用Cookies提供自动登录以及设置记忆支持。

本网站收集包括但不限于:

设备型号
浏览器内核版本
设备屏幕参数

来改善读者体验

域名

本域名 kimiblock.top 由Kimiblock Zhou 拥有.

与任何人和组织无关, 由Kimiblock个人维护和管理

评论

技术支持

本网站采用 Giscus 和 GitHub Discussions 作为评论支持.

由于众所周知的原因, GitHub在国内访问速度稍慢. 评论发布的过程可能长达1分钟. 在发表过程中请勿刷新页面.

规范

本网站在非特殊情况下不参与任何包括但不限于

List
明星, 227, 饭圈, 电视剧, 明星粉丝, 粪坑, AO3, GFW

的讨论

评论区请勿参与关于以上内容的任何讨论, 否则 Kimiblock 有权利删除评论或向 Github 举报相关不当言论

本博客所有文章除非特别申明, 均属于 CC BY-NC-SA 4.0 协议, 由 Kimiblock 创作

显示与亮度

深色模式

本网站会根据 浏览器版本 设备深色模式状态 自动启用深色模式

如果设备不支持自动切换，本网站会根据时间按需启用深色模式

兼容性

由于 Safari 自身的缺陷, 本站对 Safari 的支持可能不完全.

请使用 Chromium Firefox 访问网站

硬件加速

本站动画效果较多, 使用软件渲染可能导致卡顿, 发热等问题. 推荐启动硬件加速来优化体验.
